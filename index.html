What I actually did

I migrated our live detection pipeline off an aging edge board and onto a Jetson Orin Nano Super, then benchmarked it honestly , over the same industrial CAN bus and serial the field unit really uses, not a clean lab bench.

The dual-camera mode is where it counts. The old board choked at 7 FPS; on the Jetson the second camera costs basically nothing and both run a full 30 FPS. For a safety system, that's the difference between seeing a person and seeing them too late.

02How I work
A1

Own the whole arc

Discovery → architecture → deployment → field support. I don't hand a design over a wall. I've sourced the components, modified the PCBs, run the FAT testing, written the CE docs, and then driven to site to fix it on call.

A2

Offline-first by default

An edge box on a mine site is like a lighthouse , nobody's coming out to reset it, so it has to run itself or it's useless. I build for no cloud, bad links, dust and vibration first, and treat connectivity as a bonus.

A3

Weigh it on ROI

I built a mining-tech company to six figures, so every design gets costed, not just admired. TCO, vendor trade-offs, competitive positioning , and I can carry that same case from a field engineer up to a CIO without changing the story.
